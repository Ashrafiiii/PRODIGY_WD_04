Project 2

I have successfully built a stopwatch web application using HTML, CSS, and JavaScript. HTML was utilized to structure the elements of the application, including buttons for starting, pausing, and resetting the stopwatch, as well as displaying the time and lap times. With the implementation of JavaScript functions, users can accurately measure and record time intervals. The startStop() function enables users to start and stop the stopwatch, while the lapReset() function allows them to record lap times and reset the stopwatch when needed. The CSS styling ensures that the stopwatch is visually appealing and user-friendly. The stopwatch web application is interactive and provides a seamless experience for users to track time intervals efficiently.
